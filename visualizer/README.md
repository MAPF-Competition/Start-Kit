# Realtime Visualizer (WebGL)

This viewer replays NDJSON tick files generated by `lifelong`.

## 1. Generate ticks from the simulator

Build as usual, then run:

```bash
./build/lifelong \
  --inputFile ./example_problems/random.domain/random_32_32_20_100.json \
  --output ./output.json \
  --vizOutput ./viz_ticks.ndjson \
  --vizTickStride 1
```

Notes:
- `--vizOutput`: path to the visualizer stream file.
- `--vizTickStride N`: write every `N`-th timestep (use larger values for long runs or large teams).

## 2. Open the viewer

From repo root:

```bash
python3 -m http.server 8080
```

Then open:

`http://localhost:8080/visualizer/`

Load your `viz_ticks.ndjson` file with the file picker.

## Complete Example Commands

Run these from repo root:

```bash
cmake -B build . && cmake --build build -j32
./build/lifelong \
  --inputFile ./example_problems/random.domain/random_32_32_20_100.json \
  --output ./output.json \
  --simulationTime 500 \
  --vizOutput ./viz_ticks.ndjson \
  --vizTickStride 1
python3 -m http.server 8080
```

Then open this local URL in your browser:

`http://localhost:8080/visualizer/`

## Controls

- Drag: pan
- Mouse wheel: zoom
- `Play/Pause`: timeline playback
- `Speed`: playback rate
- Slider: scrub ticks manually
- Red agents: currently in delay
